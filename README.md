# React Portfolio App (ReactApp2)

## 👩‍💻 About
This is a multi-page portfolio website built using Next.js and React.

## 🚀 Features
- Home page introduction
- About page
- Projects page (2 sample projects)
- Contact form
- Navigation using Next.js App Router

## 🛠️ Technologies Used
- React
- Next.js
- Tailwind CSS

## 📂 Pages
- Home (/)
- About (/about)
- Projects (/projects)
- Contact (/contact)
